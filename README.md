# Modern Business Website (Static)

A modern, responsive static website designed for small businesses to showcase
their services, brand identity, and contact information.

## Live Demo

🔗 https://modern-business-website.vercel.app

## Features

- Fully responsive design (mobile, tablet, desktop)
- Clean and modern UI
- Client-focused layout and navigation
- Reusable component-based structure
- Fast performance using static rendering

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Vercel (Deployment)

## Project Purpose

This project was built as a portfolio demonstration to showcase the ability
to design and develop a client-ready static business website with a strong
focus on usability, responsiveness, and clean architecture.

## Getting Started

1. Clone the repository:
   git clone https://github.com/NirmitiMestri/modern-business-website.git

2. Install dependencies:
   npm install

3. Run the development server:
   npm run dev

4. Open http://localhost:3000 in your browser.

## Learnings

- Improved understanding of responsive layouts
- Better component structuring using Next.js App Router
- Designing UI with a client-first approach
